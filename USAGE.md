
# Bank Account System - Installation and Usage Guide

This guide explains how to set up and build the **Bank Account System** project on a fresh environment.

## Prerequisites

- **PowerShell** (Windows PowerShell or PowerShell Core)  
- Any required dependencies for the project (MSVC, git, cmake)  

---

## 1. Setup Environment

Run the one-time environment setup script:

```powershell
.\.scripts\setup-environment.ps1
```

## 2. Initializing Project

Run the project initialization script once per project:

```powershell
.\.scripts\initialize-project.ps1
```

## 3. Build Project

Whenever you want to build the project, use:

```powershell
.\.scripts\build.ps1
```
